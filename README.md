3.	Об’єкти даних 
Інформація про книги зберігається в текстовому документі «books.txt» або вводиться в stdin. Дані зберігаються у вигляді:
Автор, назва, кількість сторінок, рік видавництва, вартість.
•	Автор: ім’я та прізвище записуються латинськими літерами у даному порядку.
•	Назва: Записується латинськими літерами. Якщо складається із 2+ слів, то при введенні об’єднати їх нижнім підкресленням ‘_’, або, за потребою, дефісом  ‘-‘.
•	Кількість сторінок, рік видавництва та вартість вводиться арабськими цифрами у даному порядку.
Усю інформацію відокремлювати знаками табуляцї. 
При введенні некоректних даних, вони не будуть внесені до списку.
Приклад подачі даних про книгу:
	Andrzei Sapkowski The_Witcher:_the_Last_Wish 1993 288 119
